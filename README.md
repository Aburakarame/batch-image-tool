# batch-image-tool

生成画像を **「メタデータ削除 → 連番リネーム → リサイズ」** して一括で整える、納品・公開のための前処理ツールです。

AI画像を **SNSや支援サイトに投稿する人／画像を納品する人** が、
安全かつ手間なく画像を整えられるように作りました。

## 特長

- **メタデータ削除** — PNG/EXIF に埋め込まれたプロンプト・モデル・LoRA 情報を除去し、意図しない流出を防ぐ
- **連番リネーム** — `asset_001.png` のように3桁ゼロ埋めで統一
- **一括リサイズ** — 縦横比を保ったまま、長辺を指定サイズに収める
- **原本は無傷** — 元フォルダには触れず、`output` に出力

## 使い方

1. `batch_image_tool.py` と同じ場所に `input` フォルダを作り、画像を入れる
2. 実行する
   ```
   python batch_image_tool.py
   ```
3. `output` フォルダに「メタデータ削除＋連番＋リサイズ済み」の画像ができる

## 設定（ファイル冒頭で変更）

| 設定 | 説明 |
|---|---|
| `src` | 元画像フォルダ（初期値: `input`） |
| `dst` | 出力フォルダ（初期値: `output`） |
| `prefix` | 連番の接頭辞（`asset` → `asset_001.png`） |
| `max_size` | 長辺の最大ピクセル（縦横比はキープ） |
| `strip_metadata` | メタデータ削除の ON/OFF |

## 必要なもの

- Python 3.x
- [Pillow](https://pypi.org/project/pillow/) … `pip install Pillow`

## Windows 用（Python 不要）版

`メタデータ削除.bat` … Python を入れなくても、**ダブルクリックだけ**で使える簡易版です。

- 使い方：処理したい PNG が入ったフォルダに `メタデータ削除.bat` を置いて、ダブルクリックするだけ
- そのフォルダ内の PNG のメタデータを削除し、`1.png` `2.png` … と連番にリネームします
- Windows 標準機能だけで動くので追加インストール不要

**注意：この .bat 版は元の PNG をその場で上書きします（元のファイル名とメタデータは失われます）。必ずバックアップを取ってから使ってください。** 原本を残したい場合は、別フォルダに出力する Python 版（`batch_image_tool.py`）を使ってください。
